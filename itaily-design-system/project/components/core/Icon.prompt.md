Renders a Lucide icon by name — Itaily's icon set (2px rounded stroke).

```jsx
<Icon name="scale" size={24} />
<Icon name="file-text" color="var(--color-accent)" />
```

Requires the `lucide` UMD global on the page. Names are kebab-case Lucide names. Used inside Button (`iconLeft`/`iconRight`), IconButton, inputs, etc.
