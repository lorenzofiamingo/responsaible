Warm-surfaced container — the standard Itaily card (16px radius, 1.5px border, soft warm shadow).

```jsx
<Card elevation="md" padding="lg">…</Card>
<Card interactive onClick={open}>…</Card>
```

`elevation`: flat / xs / sm / md / lg. `padding`: none / sm / md / lg. Set `interactive` for clickable cards (lifts on hover).
