Primary action control for Itaily — use for any clickable action; one `primary` per view.

```jsx
<Button variant="primary" size="md">Ask Itaily</Button>
<Button variant="secondary" iconLeft={<Icon name="file-text" />}>View source</Button>
<Button variant="ghost" size="sm">Cancel</Button>
```

Variants: `primary` (terracotta fill — the main CTA), `secondary` (outlined, neutral), `ghost` (text only), `inverse` (white, for dark surfaces). Sizes `sm | md | lg`. Props: `iconLeft`, `iconRight`, `full`, `disabled`. Font is Space Grotesk medium with tight tracking — matches the wordmark.
