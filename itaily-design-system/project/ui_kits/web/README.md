# Itaily — Web UI kit

The **marketing / pitch landing page** for Itaily (Hack the Law @ Cambridge).

## Run
Open `index.html` (loads `_ds_bundle.js`, Lucide, React + Babel, then `Landing.jsx`).

## Sections
- **Nav** — wordmark, links, Sign in (→ app kit), Request access.
- **Hero** — "Italian law, decoded." headline + a *live cited-answer demo card* (click a citation to highlight its source).
- **How it works** — three steps on the warm-white band.
- **The corpus** — primary-source coverage grid + "updated daily".
- **CTA** — inverted ink section, light wordmark, "Ask. Cite. Comply."
- **Footer**.

## Composes
`Button`, `Icon`, `Badge`, `Card`, `Citation`, `SourceCard`, `ConfidenceMeter`.

> Recreation note: there is no existing marketing site — this realises the pitch using the brand foundations. Copy is illustrative.
